🛡️ nmap_projem: Cyber Security Suite
nmap_projem, siber güvenlik araştırmacıları ve sistem yöneticileri için geliştirilmiş, Python ve PyQt5 tabanlı görsel bir ağ analiz ve port tarama arayüzüdür. Nmap ve NBTScan araçlarının gücünü, modern ve kullanıcı dostu bir "Hacker" temalı kontrol panelinde birleştirir.

✨ Özellikler
Gelişmiş Port Analizi: Hızlı tarama, tam port taraması, servis/versiyon analizi ve işletim sistemi tespiti seçenekleri.

Ağ Keşfi (IP Bulma): Belirli bir ağ bloğundaki aktif cihazları hızla tespit etme.

Gerçek Zamanlı Çıktı: Tarama sonuçlarını anlık olarak terminal benzeri bir ekranda izleme.

Raporlama Sistemi: Yapılan tüm taramaları tek bir merkezde toplama ve .txt formatında dışa aktarma.

İlerleme Takibi: Dinamik ProgressBar ile işlemin durumunu görsel olarak takip etme.

🚀 Kurulum
Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyin:

Gereksinimleri Yükleyin:
Sisteminizde nmap ve nbtscan araçlarının yüklü olduğundan emin olun. Ardından gerekli Python kütüphanelerini yükleyin:

Bash
pip install PyQt5
Projeyi Çalıştırın:

Bash
python main.py
